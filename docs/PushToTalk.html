PushToTalk tool for remote conference chats

I'm working remotetly for last 5 years and this is the tool that makes conferences nice for everyone. When my conference application starts PushToTalk activate push-to-talk mode. In this mode I'm muted and I need to press fn key to temporaty unmute myself. Also I have keybindings to easyly switch from push-to-talk to release-to-talk mode, in case I'm active speaker.

PushToTalk has menubar with icons so when you see red circle - you shod know you're unmuted.

This is my config: spoon.SpoonInstall:andUse("PushToTalk", {start = true, config = { app_switcher = { ['zoom.us'] = 'push-to-talk' }}}) and binding for this function() spoon.PushToTalk.toggleStates({'push-to-talk', 'release-to-talk'}) end
